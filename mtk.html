Soal 3

Tuliskan algoritma untuk menghitung faktorial dari sebuah bilangan.

    
    // Kode JavaScript sederhana
    function hitungFaktorial(n) {
        if (n === 0) {
            return 1;
        }
        return n * hitungFaktorial(n - 1);
    }

Hasil dari 5 + 7 adalah 12.
